# 六边形物理模拟游戏

一个基于现代Web技术的实时物理模拟演示，展示了在旋转六边形容器中的小球物理运动。

## 🎮 游戏特性

### 核心功能
- **旋转六边形容器**：可视化的正六边形容器，能够绕其中心点连续旋转
- **物理小球**：具备完整物理特性的小球，包括重力、摩擦力和弹性
- **精确碰撞检测**：与六边形所有六条边的精确碰撞检测
- **实时物理模拟**：60fps流畅的物理计算和渲染

### 物理特性
- ✅ **重力系统**：可调节的向下重力加速度
- ✅ **摩擦力**：影响球的运动衰减
- ✅ **弹性碰撞**：可调节的弹性系数控制碰撞后的能量保持
- ✅ **空气阻力**：模拟真实的空气阻力效果
- ✅ **速度限制**：防止物理计算溢出的最大速度限制

### 视觉效果
- 🎨 **现代UI设计**：深色主题，渐变色彩
- 🌟 **轨迹追踪**：显示小球的运动路径
- 💫 **粒子效果**：小球具有渐变和高光效果
- 📱 **响应式设计**：适配不同屏幕尺寸

## 🛠️ 技术实现

### 技术栈
- **前端框架**：Preact (轻量级React替代)
- **构建工具**：Vite (快速开发和构建)
- **语言**：TypeScript (类型安全)
- **渲染**：Canvas 2D API (高性能图形渲染)
- **样式**：CSS3 (现代样式和动画)

### 核心算法
- **向量数学**：完整的2D向量运算库
- **碰撞检测**：点与线段的精确碰撞算法
- **物理模拟**：基于牛顿力学的运动计算
- **坐标变换**：旋转坐标系下的法向量计算

### 架构设计
```
src/
├── utils/
│   ├── Vector2D.ts      # 2D向量数学库
│   ├── Physics.ts       # 物理引擎核心

│   ├── Ball.ts          # 小球实体类
│   └── GameEngine.ts    # 游戏引擎和渲染系统
├── components/
│   ├── ControlPanel.tsx # 控制面板组件
│   └── HexagonGame.tsx  # 主游戏组件

```

## 🎯 操作说明

### 基本控制
- **鼠标点击**：在六边形内部点击重新定位小球
- **空格键**：暂停/继续游戏
- **R键**：重置游戏状态

### 参数调节
通过右侧控制面板可以实时调节：

#### 物理参数
- **重力**：调节重力加速度 (0-1000)
- **摩擦力**：调节摩擦系数 (0.9-1.0)
- **弹性系数**：调节碰撞后的能量保持 (0-1.0)
- **空气阻力**：调节空气阻力系数 (0.99-1.0)
- **最大速度**：限制小球的最大速度 (100-2000)

#### 六边形控制
- **旋转速度**：调节六边形的旋转速度 (-2 到 2 弧度/秒)

#### 小球设置
- **半径**：调节小球大小 (5-30像素)
- **质量**：调节小球质量 (0.1-5.0)
- **颜色**：自定义小球颜色
- **轨迹长度**：调节轨迹显示长度 (0-100)
- **显示轨迹**：开关轨迹显示

## 🔬 物理原理

### 碰撞检测算法
1. **点到线段距离计算**：使用向量投影计算球心到六边形边的最短距离
2. **法向量计算**：根据六边形当前旋转角度计算正确的法向量
3. **穿透检测**：检测小球是否穿透六边形边界
4. **位置修正**：将小球推出碰撞区域

### 物理力计算
1. **重力**：恒定的向下加速度
2. **摩擦力**：与速度成正比的阻力
3. **空气阻力**：速度的平方项阻力
4. **弹性碰撞**：基于动量守恒的反弹计算

### 数值积分
使用欧拉积分方法更新物体位置和速度：
```
velocity = velocity + acceleration * deltaTime
position = position + velocity * deltaTime
```

## 🎨 自定义和扩展

### 添加新的物理效果
1. 在 `Physics.ts` 中添加新的力计算方法
2. 在 `GameEngine.ts` 的更新循环中应用新的力
3. 在 `ControlPanel.tsx` 中添加对应的控制界面

### 修改渲染效果
1. 在 `Ball.ts` 中修改小球的渲染方法
2. 在 `Hexagon.ts` 中修改六边形的渲染样式
3. 在 `app.css` 中调整UI样式

### 性能优化建议
- 使用 `requestAnimationFrame` 确保流畅渲染
- 限制 `deltaTime` 避免大的时间跳跃
- 使用对象池减少垃圾回收
- 优化碰撞检测算法的计算复杂度
